ViewAnalysis

Display analysis on the provided context.

Intent Name

ViewAnalysis

Display Name

View Analysis

Possible Contexts

  • Instrument
  • Organization
  • Portfolio

Example

const instrument = {
    type: 'fdc3.instrument',
    name: 'International Business Machines',
    id: {
        ticker: 'ibm'
    }
}

fdc3.raiseIntent('ViewAnalysis', instrument)
